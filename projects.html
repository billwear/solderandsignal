Instrumentation

First thing we need to do is assemble a set of basic instruments we can use to build more circuits. Ultimately, you'll want to be able to test components and circuits to verify values, check tolerances, and spot bad parts. And you can learn most of your basic analog electronics more quickly this way.

There are plenty of speciality tools and kits out there on the World Wild Web, but I'm trying to build them from things I already have available on my farm. I'll try to make enclosures as easy to build as possible, but you may want to consider buying project boxes of various sizes. Woodworking skills are helpful, but not essential here.
