Basic DC voltmeter

⚠️ DIY projects involve real tools and voltages. Read the safety disclaimer.

What this project is about

We're going to build a basic DC voltmeter, starting on the breadboard, and then building it into a homemade case that you can carry around with you. This will be a a direct-coupled amplifier voltmeter built with transistors and a moving-coil meter (PMMC). The main stages will be:

  • An ttenuator: Adjusts input so the meter remains accurate across ranges.
  • A two-transistor amplifier: Boosts the micro-voltage signal without needing power-hungry op-amps.
  • Zero adjustment potentiometer: Calibrates the meter to read zero when no voltage is applied.
  • Meter movement (right): A needle display that visually indicates voltage.

    • This circuit is elegant, low cost, and doesn’t need a microcontroller. It’s wonderful for making voltage visible in an analog, charming way—perfect for reflective experimenters.
